# Система учета сотрудников Университета "Синергия"

## Описание проекта
Программа для учета и фильтрации сотрудников административного персонала Университета "Синергия" по стажу работы.

## Функциональность
- Ввод данных о сотрудниках университета
- Расчет стажа работы каждого сотрудника
- Фильтрация сотрудников по минимальному стажу
- Вывод информации в удобном формате
