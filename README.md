# Система учета сотрудников Университета "Синергия"

## Описание проекта
Программа для учета и фильтрации сотрудников административного персонала Университета "Синергия" по стажу работы.

## Функциональность
- Ввод данных о сотрудниках университета
- Расчет стажа работы каждого сотрудника
- Фильтрация сотрудников по минимальному стажу
- Вывод информации в удобном формате

## Структура проекта
synergy_staff_system/
├── university_staff.h # Заголовочный файл класса
├── university_staff.cpp # Реализация методов класса
├── main.cpp # Основная программа
└── README.md # Документация

## Класс UNIVERSITY_STAFF
Содержит поля:
- Фамилия и инициалы сотрудника
- Должность в университете
- Зарплата
- Год приема на работу

Методы:
- Конструкторы (по умолчанию и с параметрами)
- Деструктор
- Геттеры и сеттеры для всех полей
- Расчет стажа работы
- Вывод информации

## Как скомпилировать и запустить

### Для Linux/macOS:
g++ -o synergy_staff main.cpp university_staff.cpp
./synergy_staff

### Для Windows (MinGW):
g++ -o synergy_staff.exe main.cpp university_staff.cpp
synergy_staff.exe

Для Visual Studio:
Создайте новый проект C++

Добавьте все три файла в проект

Скомпилируйте и запустите
